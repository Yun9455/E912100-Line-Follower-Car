## 專案說明
本專案為 NodeMCU-32S/ESP32 兩輪5路循跡自走車之核心控制程式碼。
包含以下實作特點：
1. 循跡控制：導入模組化狀態機與Pure Pursuit運動控制。針對實體機構18公分之前懸長度，優化Lookahead Distance參數以提升高速穩定性。
2. 直角防呆機制：實作「歷史加權趨勢(Weighted Trend)」演算法，防堵過彎時感測器瞬間失訊導致的轉向誤判的現象。
3. 安全狀態機：具備完整的EBS(緊急制動)、TSMS 與VLS啟動邏輯，確保競賽安全性。
